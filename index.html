The DFS Algorithm

1. Depth First Search: Explores a branch as deep as possible before backtracking.

2. Implicit Stack: Uses the system's Call Stack via recursion to remember the path.

3. Time Complexity: O(V + E) where V is vertices and E is edges.

4. Space Complexity: O(V) due to the recursive stack depth.
