How DFS Works on Trees

1. Start at the root node.

2. Visit the node and push it onto the call stack.

3. Go to the leftmost unvisited child — dive deep.

4. If no children left → backtrack (pop from stack).

5. Repeat until all nodes are visited or the target is found.

Key: DFS goes as DEEP as possible before going WIDE.
